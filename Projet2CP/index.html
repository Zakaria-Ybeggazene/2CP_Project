  • Recherche

  • Comment puis-je rechercher un étudiant ?

    C'est assez simple de trouver un étudiant, tout ce que vous devrez faire est :

    1. Dans le menu qui se trouve à gauche dans la fenêtre principale, cliquez sur "Etudiant".
    2. Un nouvel onglet apparaîtra dans l'espace de travail.
    3. Remplissez les champs des filtres que vous voulez appliquer puis cliquez sur "Filtrer".

    Les résultats correspondants vont apparaître dans le tableau en dessous (chaque ligne représente un résultat).

  • Comment puis-je rechercher une promotion ?
    1. Dans le menu qui se trouve à gauche dans la fenêtre principale, cliquez sur "Promotion".
    2. Un nouvel onglet "Promotion" apparaîtra dans l'espace de travail.
    3. Dans les combobox marqués "Année" et "Niveau", choisissez l'année et le niveau désirés puis cliquez sur "Filtrer".

    La recherche affiche deux tableaux. Le premier se trouvant à gauche contient les matières suivies par cette promotion accompagnées de leurs moyennes. Le deuxième contient la liste des étudiants de la promotion ordonnés selon leurs moyennes.

  • Comment afficher le classement de fin de cursus ?

    C'est la même procédure que celle de la recherche d'une promotion (paragraphe précédent), sauf que vous devrez choisir l'option "SI3", "SIQ3" ou "SI3 & SIQ3" dans combobox marqué "Niveau". Filtrez et vous aurez le classement selon la moyenne des quatres années dans le même onglet.

  • Comment afficher les détails d'un étudiant ?
    Il existe 2 (deux) manières pour le faire :

    Rechercher l'étudiant (voir le paragraphe "Comment puis-je rechercher un étudiant ?") puis cliquer sur le bouton (⋮) qui se trouve dans la dernière colonne (Détails) sur la ligne de cet étudiant.

    Ou bien

    Après avoir trouvé une promotion, si les informations d'un étudiant vous intéressent, allez dans la liste des étudiants de cette promotion puis cliquez sur le même bouton décrit dans la première méthode pour leur accéder.

  • Pourquoi le bouton "Réinitialiser” ?

    Vous allez trouver ce bouton dans les deux onglets "Etudiant" et "Promotion", il sert à vider les champs des filtres et réinitialiser le(s) résultat(s) en cas où l'utilisateur veut effectuer une nouvelle requête.

  • Etats de sortie

  • Comment générer une attestation ?

    D'abord, vérifiez si votre système dispose de la version de Crystal Reports recommandée dans le fichier d'installation, si elle est installée effectuez ce qui suit :

    1. Recherchez l'étudiant désiré (voir "Comment puis-je rechercher un étudiant ?" dans la section "Recherche").
    2. Accédez à ses détails (voir "Comment afficher les détails d'un étudiant ?" dans la section "Recherche").
    3. Dans l'onglet de cet étudiant, cliquez sur le bouton "Générer Attestation".

    Une fenêtre de Crystal Reports va s'ouvrir pour afficher l'attestation, vous pouvez l'imprimer à partir de cette fenêtre.

  • Comment générer un relevé de notes d'un étudiant ?

    D'abord assurez-vous que votre système dispose de la version du Crystal Reports recommandée dans le fichier d'installation, si elle est installée effectuez ce qui suit :

    1. Ouvrez l'onglet de l'étudiant (voir les étapes 1 & 2 dans le 1er paragraphe de cette section).
    2. Dans l'onglet de cet étudiant, précisez l'année dans le combobox libellé "Année" en bas à gauche puis cliquez sur le bouton "Générer relevé de notes".

    Une fenêtre de Crystal Reports va s'ouvrir pour afficher le relevé de notes, vous pouvez l'imprimer à partir de cette fenêtre.

  • Comment générer le relevé global d'un étudiant ?

    D'abord assurez-vous que votre système dispose de la version du Crystal Reports recommandée dans le fichier d'installation, si elle est installée effectuez ce qui suit :

    1. Ouvrez l'onglet de l'étudiant (voir les étapes 1 & 2 dans le 1er paragraphe de cette section).
    2. Dans l'onglet de cet étudiant, cliquez sur le bouton "Générer Relevé Global".

    Si l'étudiant a complété ses 5 années alors une fenêtre de Crystal Reports va s'ouvrir pour afficher le relevé global que vous pouvez l'imprimer à partir de cette fenêtre. S'il n'a pas complété ses 5 années vous aurez un message vous indiquant qu'il doit les avoir complétées.

  • Comment générer un PV de délibération ?

    D'abord assurez-vous que votre système dispose de la version du Crystal Reports recommandée dans le fichier d'installation, si elle est installée effectuez ce qui suit :

    1. Recherchez la promotion (voir "Comment puis-je rechercher une promotion ?" dans la section "Recherche").
    2. Cliquez sur le bouton "Générer PV".

    Une fenêtre de Crystal Reports va s'ouvrir pour afficher le PV de délibération, vous pouvez l'imprimer à partir de cette fenêtre.

  • Statistiques

  • Comment afficher les statistiques ?

    L'onglet des statistiques est accessible pour tout utilisateur s’il clique sur le bouton "Statistiques" dans le menu à gauche.

    Il peut choisir le type des statistiques qu'il veut afficher avec les flèches qui se trouvent à droite et à gauche du titre "Statistiques (générales/promotion/matière)". Généralement l'utilisateur doit spécifier quelques détails dans les listes déroulantes disponibles pour afficher les statistique (exemple : Année et Niveau pour les statistiques d'une promotion).

    Les statistiques d'une promotion peuvent être accédées à partir de l'onglet de la recherche d'une promotion, le bouton "Statistiques" va ouvrir et initialiser l'onglet et dans ce cas, l'utilisateur n'est amené à spécifier quoi que ce soit.

  • Base de données

  • Accéder au fichier Access

    D'abord, vous devez vous connecter au mode administrateur pour pouvoir utiliser cette fonctionnalité (voir la section "Confidentialité"). Ensuite suivez les étapes suivantes :

    1. Cliquez sur "Réglages" puis "Base de données".
    2. Cliquez sur "Ouvrir".

    Si un problème est survenu et vous ne pouviez pas ouvrir la BDD, vous pouvez le faire manuellement en suivant ce qui suit :

    1. Dans la même fenêtre, cliquez sur "Afficher" et copiez le mot de passe de la BDD.
    2. Ouvrez le fichier Access "db.accdb" dans le dossier "C:\\Utilisateurs\"nom de l'utilisateur courant"\AppData\Roaming\" et entrez le mot de passe.
  • Modifier les informations d'un étudiant dans la BDD

    Premièrement, vous devez vous connecter au mode administrateur pour pouvoir exploiter ce privilège (voir la section "Confidentialité"). Ensuite :

    1. Ouvrez l'onglet de l'étudiant (voir la section "Recherche").
    2. Cliquez sur "Modifier".
    3. Modifiez les champs que vous désirez changer (cliquez sur un champ pour modifier son contenu).
    4. Cliquez sur "Sauvegarder" pour enregistrer les modifications.
  • Recréer la base de données

    Si vous voulez, créer la base de données (le fichier Access) à nouveau à partir des fichiers Excel (après avoir corriger les données probablement), suivez la procédure suivante :

    1. Connectez vous au mode administrateur si ce n’est pas encore fait (voir la section "Confidentialité").
    2. Ouvrez la fenêtre des réglages par un seul clic sur le bouton "Réglages" dans le menu principal.
    3. Allez vers "Base de données".
    4. Cliquez sur "Supprimer" puis sur "Oui".

    Cette procédure va vous rediriger vers la fenêtre de l'importation des fichiers Excel ou vous devriez le refaire dans l'ordre spécifié. Après la validation des 4 fichiers Excel, insérez et confirmez les nouveaux mots de passe pour le simple utilisateur et l'admin et attendez pendant que la fin de la migration.

    Vous pouvez toujours consulter le manuel d'utilisation pour une meilleure illustration des étapes.

  • Confidentialité

  • Mode administrateur

    Si l'application vous exige d'être connectés en tant qu'administrateur vous devez juste aller au menu principal et cliquer sur "Mode Administrateur" et entrer votre mot de passe.

    Pour se déconnecter de ce mode, cliquez juste sur "Se Déconnecter" dans le menu principal.

  • Changer un mot de passe

    Premièrement, vous devez vous connecter au mode administrateur pour pouvoir utiliser cette fonctionnalité (paragraphe précédent). Après :

    1. Dans le menu principal, cliquez sur le bouton "Réglages".
    2. Choisissez le type d'utilisateur (Utilisateur/Admin) pour lequel vous changer de mot de passe.
    3. Dans la fenêtre des réglages, entrez le mot de passe actuel, le nouveau mot de passe et cliquez sur "Modifier" pour enregistrer.
